# Kalman Filtering and Sensor Fusion Simulation

## Setup Instruction

### Install the dependencies

 ```
 sudo apt install libeigen3-dev libsdl2-dev libsdl2-ttf-dev
 ```

### Setup the cmake build

```
mkdir build
cd build
cmake ../
```


### Compile the code
```
make
```

### Run the estimation simulator
```
./AKFSF-Simulation
```
